## 💖 Git Flow 
- `main` : 프로덕트를 배포하는 브랜치
- `develop`: 프로덕트 배포 전 기능을 개발하는 브랜치
- `feature`: 단위 기능을 개발하는 브랜치로 단위 기능 개발이 완료되면 develop 브랜치에 merge ex) feat/#이슈번호
